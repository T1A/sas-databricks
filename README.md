## About
This project contains T1A's public accelerators as well as code examples on how Databricks and SAS can be integrated.


1. SAS-Libraries-Examples a guide an code sample on how to connect SAS 9.4 to Databricks over  SAS/ACCESS Interface to JDBC or SAS/ACCESS Interface to Spark  https://github.com/T1A/sas-databricks/tree/main/SAS-Libraries-Examples
2. SAS-DBR-Custom-Integration a code sample of custom integration a-la cloudfetch but over SSH https://github.com/T1A/sas-databricks/tree/main/SAS-DBR-Custom-Integration
